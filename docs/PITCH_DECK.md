```
                  ┌─────────────┐

                  │   Redis     │

```

### Technology Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express.js, Socket.IO
- **ML Engine**: Python, Flask, Isolation Forest
- **Database**: MongoDB, Redis
- **Infrastructure**: Docker, Nginx, SSL/TLS
